🧮 Mathematical Foundation

$$q(x_t | x_{{t-1}}) = \mathcal{N}(x_t; \sqrt{1-\beta_t} x_{{t-1}}, \beta_t I)$$
Intuition: Gradually add Gaussian noise over T timesteps until data becomes pure noise.
$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1-\bar{\alpha}_t} \epsilon$$
where \(\alpha_t = 1 - \beta_t\), \(\bar{\alpha}_t = \prod_{s=1}^t \alpha_s\), \(\epsilon \sim \mathcal{N}(0,I)\)
  • Markovian: Each step only depends on previous step
  • Gaussian: Each transition is a Gaussian distribution
  • Closed form: Can sample x_t directly from x_0
$$p_\theta(x_{{t-1}} | x_t) = \mathcal{N}(x_{{t-1}}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$
Goal: Learn to reverse the noise addition process.
$$L_\text{simple} = \mathbb{E}_{t,x_0,\epsilon}[||\epsilon - \epsilon_\theta(\sqrt{\bar{\alpha}_t} x_0 + \sqrt{1-\bar{\alpha}_t} \epsilon, t)||^2]$$
Intuition: Train neural network to predict the noise that was added.
def linear_beta_schedule(timesteps):
    beta_start = 0.0001
    beta_end = 0.02
    return torch.linspace(beta_start, beta_end, timesteps)

def cosine_beta_schedule(timesteps, s=0.008):
    steps = timesteps + 1
    x = torch.linspace(0, timesteps, steps)
    alphas_cumprod = torch.cos(((x / timesteps) + s) / (1 + s) * torch.pi * 0.5) ** 2
    alphas_cumprod = alphas_cumprod / alphas_cumprod[0]
    betas = 1 - (alphas_cumprod[1:] / alphas_cumprod[:-1])
    return torch.clip(betas, 0.0001, 0.9999)

🎯 Interview Q&A

Q1: Explain the intuition behind diffusion models. Why do they work?
A: Gradually transform data to noise, learn to reverse, score matching, stable training, high quality.
Q2: Compare diffusion models to GANs. When would you choose each?
A: Diffusion: stable, high quality, diverse, slow. GANs: fast, efficient, unstable, mode collapse.
Q3: How would you make diffusion model sampling faster?
A: DDIM, ODE solvers, distillation, latent diffusion, consistency models.
Q4: Explain classifier-free guidance and why it's important.
A: No separate classifier, single model, stable, better conditioning, trade-off between conditioning and diversity.
