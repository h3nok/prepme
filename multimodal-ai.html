🔑 Key Concepts

  • Modality: A type of data (e.g., image, text, audio, video, 3D)
  • Alignment: Mapping between modalities (e.g., image-text pairs)
  • Fusion: Combining information from multiple modalities
  • Cross-modal transfer: Using one modality to improve another

🏗️ Architectures

  • Encoders: Separate neural networks for each modality (e.g., CNN for images, Transformer for text)
  • Fusion Module: Combines features (concatenation, attention, gating, etc.)
  • Joint Embedding: Project all modalities into a shared space
  • Cross-Attention: Attend to one modality while processing another (e.g., text attends to image regions)
# Example: Cross-modal Transformer
class CrossModalTransformer(nn.Module):
    def __init__(self, image_dim, text_dim, hidden_dim):
        super().__init__()
        self.image_proj = nn.Linear(image_dim, hidden_dim)
        self.text_proj = nn.Linear(text_dim, hidden_dim)
        self.transformer = nn.Transformer(hidden_dim)
    def forward(self, image_feat, text_feat):
        img = self.image_proj(image_feat)
        txt = self.text_proj(text_feat)
        # Concatenate and process
        x = torch.cat([img, txt], dim=1)
        return self.transformer(x)

🔗 Fusion Strategies

  • Early Fusion: Concatenate raw or low-level features, then process jointly
  • Late Fusion: Process each modality separately, then combine predictions
  • Cross-modal Fusion: Use attention to fuse at multiple levels
  • Gated Fusion: Learn weights for each modality dynamically
# Example: Gated Fusion
class GatedFusion(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.gate = nn.Linear(dim * 2, 1)
    def forward(self, x1, x2):
        gate = torch.sigmoid(self.gate(torch.cat([x1, x2], dim=-1)))
        return gate * x1 + (1 - gate) * x2

🌍 Applications

  • Image captioning (e.g., BLIP, Flamingo)
  • Visual question answering (VQA)
  • Text-to-image generation (DALL-E, Stable Diffusion)
  • Speech-to-text, video understanding, robotics
  • Multimodal retrieval and search

🧪 Research Frontiers

  • Scaling to more modalities (video, 3D, sensor data)
  • Unsupervised and self-supervised multimodal learning
  • Multimodal large language models (e.g., GPT-4V, Gemini, LLaVA)
  • Alignment and grounding (e.g., aligning text to image regions)
  • Efficient cross-modal retrieval and generation

🎯 Interview Q&A

Q1: What is early vs. late fusion in multimodal models?
A: Early fusion combines features before processing; late fusion combines outputs after separate processing.
Q2: How does cross-attention enable multimodal learning?
A: Allows one modality (e.g., text) to attend to features of another (e.g., image regions), enabling richer interactions.
Q3: Name a challenge in aligning modalities.
A: Data may be weakly aligned (e.g., image and caption not perfectly matched), requiring robust alignment techniques.
