What is an LLM?

Large Language Models are deep neural networks trained on massive text corpora to generate, summarize, and reason about language.

Key features:
  • Billions of parameters
  • Transformer-based architecture
  • Pre-training + fine-tuning

Applications

  • Text generation (e.g., ChatGPT, Claude)
  • Summarization
  • Translation
  • Code generation
