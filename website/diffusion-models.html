What is a Diffusion Model?

Diffusion models are a class of generative models that learn to reverse a gradual noising process, producing high-quality samples from noise.

$$q(x_t|x_{t-1}) = \mathcal{N}(x_t; \sqrt{1-\beta_t}x_{t-1}, \beta_t I)$$
Key idea: The model learns to denoise step-by-step, eventually generating realistic data (e.g., images, audio) from pure noise.

Applications

  • Image synthesis (e.g., DALL·E 2, Stable Diffusion)
  • Audio generation
  • Scientific data simulation
